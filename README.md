# AI Chatbot 

A conversational AI chatbot powered by the OpenAI API. 

##  Overview
This project is an intelligent chatbot designed to interact with users using natural language processing. It leverages OpenAI's models to provide helpful, context-aware responses.

## Features
* **OpenAI Integration:** Utilizes the OpenAI API for generating dynamic responses.
* **Secure Setup:** Uses environment variables to securely manage API keys.
* **Extensible:** Ready to be integrated into web interfaces or run directly in the terminal.

## Prerequisites
Before running this project, ensure you have the following installed:
* Python 3.8+
* An active [OpenAI API Key](https://platform.openai.com/api-keys)
